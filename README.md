# md5-cli

Dead simple. You type:

```bash
md5-cli "Your string"
```

and it prompts the result: `319557b02e00284cd64cde1c4bbfe2e5`

Works also on files:
```bash
md5-cli "index.js"
24630afcf044d185d3acce04ccc4d4fc
```

# Installation
```
npm install -g md5-cli
```
